## Task overview

Create a small todo app using Angular and Angular Material.

Priorities:
- Use [pnpm](https://pnpm.io/) (not npm!) as the package manager,
- create a lazy loaded `TodoModule` Angular module,
- use the Angular router,
- write your code clean.
- Expected MVP:
  - List todos,
  - add,
  - edit,
  - delete and
  - mark todos as done.
  - Finished todos should look different from open todos

## Data Storage
Feel free to decide how you store the data. For example:
- in memory,
- local storage,
- session storage or
- using an api.

## How to start
- Create an Angular application using the Angular CLI
  - Add the Angular Router and use SCSS for styling
- Add Angular Material with a default theme.
- Code!
